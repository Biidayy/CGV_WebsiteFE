1. Introduction

This project is a front-end clone of the CGV Cinemas Vietnam website. Users can browse movies and cinemas, open movie details with trailers, save favorites, and complete a mocked ticket-booking flow (seat selection) stored in the browser.

The site is built with HTML, CSS, and JavaScript only. There is no real login, payment gateway, or backend API.

2. System Requirements

3. How to Access the Website

3.1 Published site (recommended)

  1. Open a web browser.
  2. Go to: https://biidayy.github.io/CGV_WebsiteFE/
  3. You will be redirected to the home page under /cgv/.

Direct home URL: https://biidayy.github.io/CGV_WebsiteFE/cgv/

3.2 GitHub repository

  1. Open https://github.com/Biidayy/CGV_WebsiteFE
  2. Confirm the repository visibility is Public.
  3. Browse source files in the cgv/ folder.

3.3 Run locally

  1. Clone or download the repository.
  2. Serve the project folder with a static server, for example:
cd CGV_WebsiteFE
python -m http.server 8765
# then open http://127.0.0.1:8765/cgv/

4. Features Overview

Main pages

  • Home — hero banner, movie carousel, events
  • Now showing / Coming soon — lists with search & filters
  • Cinemas — all cinemas, special formats, 3D
  • Movie detail — info, formats, trailer, booking link
  • Book tickets (mock) — session & seat selection
  • My tickets / Favorites — saved tickets and liked movies

Extra features

  • Search/filter by title, genre, format, rating
  • Favorites via browser localStorage
  • Mock seat booking + toast messages
  • Back-to-top button and responsive mobile menu

5. Step-by-Step Instructions

5.1 Browse movies on the home page

  1. Open the published website.
  2. View the hero slider and Movie Selection rail.
  3. Click a movie poster or Xem chi tiết to open movie detail.

5.2 Search and filter movies

  1. Open PHIM → Phim Đang Chiếu (or Phim Sắp Chiếu).
  2. Type a title/genre in the search box.
  3. Optionally choose Genre / Format / Rating filters.
  4. Click Reset to clear filters.

5.3 View movie details and trailer

  1. Open any movie detail page.
  2. Review synopsis, runtime, rating, and formats.
  3. Play the trailer when available.
  4. Use the buy-ticket action to continue to booking.

5.4 Book a ticket (mock flow)

  1. On the booking page, select available session options.
  2. Choose one or more empty seats on the seat map.
  3. Confirm booking — the ticket is stored locally (no real payment).

5.5 Manage favorites and tickets

  1. Click the favorite (heart) control on a movie.
  2. Open YÊU THÍCH or VÉ CỦA TÔI from the top menu.
  3. Review saved items. Clearing browser data resets them.

5.6 Explore cinemas

  1. Open RẠP CGV → Tất Cả Các Rạp.
  2. Visit Rạp Đặc Biệt and Rạp 3D for format info.

7. Known Limitations

  • Frontend-only mock: no real login, payment, or server API.
  • Tickets and favorites exist only in browser localStorage.
  • Some menu items (Cultureplex, member login, etc.) are placeholders.
  • Movie/cinema data is static demo data in js/data.js.
  • External image assets require an internet connection.
  • This is a coursework demo, not an official CGV product.
